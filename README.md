# projects_school21

Выполненные мной проекты из Школы 21

DO1_Linux — просто конспект того, как я работала в виртуальной машине на серверной Ubuntu 20.04, и изученного мной в процессе материала.
C3_SimpleBashUtils — упрощённые реализации Linux-утилит `cat` (поддержка флагов b, e, n, s, t, v) и `grep` (поддержка флагов e, i, v, c, l, n, h, s, f, o)
C2_s21_stringplus — реализация основных функций библиотеки `string.h`, а также функции `sprintf` с поддержкой полного форматирования с учетом флагов, ширины, модификаторов и типов преобразования (спецификаторы: c, d, f, s, u, %, g, G, e, E, x, X, o, p; флаги: -, +, (пробел), #, 0; ширина: (число), `*`; точность: .(число), .*; длина: h, l, L)
C6_s21_matrix — реализация библиотеки `matrix.h`, для операций с матрицами
